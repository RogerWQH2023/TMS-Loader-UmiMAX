# README

## 项目启动方式

- 安装依赖

```
pnpm install
```

- 运行项目

```
pnpm start
```

## 项目说明

- 本项目基于UmiMax搭建

- 本项目通过[umi-cesium-plugin](https://github.com/cnyballk/umi-cesium-plugin)以umi插件形式配置cesium。

- TMSLayer部分使用JS编写，MineLayer部分使用TS编写

`@umijs/max` 模板项目，更多功能参考 [Umi Max 简介](https://umijs.org/docs/max/introduce)
